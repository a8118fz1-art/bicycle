================================

- Fix 3-STEP TEST execution procedure only.

Fixed:
- Each STEP timer is fixed to 10 seconds.
- STEP indicator changes according to the current running STEP.
- STEP 1 / STEP 2 / STEP 3 each records max RPM.
- STEP 2 KP is calculated from STEP 1 max RPM.
- STEP 3 KP is calculated from STEP 2 max RPM.
- STEP 3 completion automatically switches to 3-STEP TEST RESULT page.

STEP 2 KP rule:
- STEP1 max RPM >= 180: base KP + 3
- STEP1 max RPM >= 150: base KP + 2
- STEP1 max RPM < 150: base KP + 1

STEP 3 KP rule:
- STEP2 max RPM >= 180: STEP2 KP + 4
- STEP2 max RPM >= 150: STEP2 KP + 3
- STEP2 max RPM >= 130: STEP2 KP + 2
- STEP2 max RPM < 130: STEP2 KP + 1
